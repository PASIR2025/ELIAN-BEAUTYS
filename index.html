Generador automático:
Cuando eliges un producto, país y plan, la app arma un mensaje con precio + método de pago + entrega + agradecimiento. Los textos vienen ya cargados para tus países y luego tú solo editas lo que necesites.
Variables automáticas:
Puedes usar estas variables dentro de cualquier mensaje: {cliente}, {producto}, {pais}, {precio}, {pago}, {plan}, {incluye}, {entrega}, {gracias}, {extra}.

🧠 Datos maestros" id="sec-maestro"
Aquí editas una sola vez precios, pagos y textos base. Luego el generador automático usa estos datos para todos los mensajes.
⚙️ Administrar estructura" id="sec-estructura"
Puedes quitar cualquier país, categoría, plan o producto desde los botones de cada chip. Si quitas uno, no se borran automáticamente los mensajes ya creados; solo sale de la estructura principal.
